## ✨ 核心功能

LInAI是一款集**数据获取-模型训练-即时部署**于一体的本地化AI工具，旨在降低AI应用门槛：
- 自动从网络采集高质量训练数据
- 本地自动化模型训练流程，无需专业知识
- 训练完成后即时加载模型，开箱即用
